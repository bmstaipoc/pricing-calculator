# Data Module Pricing Calculator

A comprehensive pricing calculator for estimating costs of data platform initiatives including ingestion, dataset management, and BI reporting components.

## Features

- **Three Core Modules**: Ingestion, Datasets, BI Reports
- **Complexity-Based Pricing**: Simple, Standard, Complex, Enterprise tiers
- **Flexible Customization**: Add-ons and optional features
- **Real-time Calculation**: Instant cost updates
- **Export Capabilities**: Generate quotes and reports
- **Migration Support**: Special pricing for tech migrations

## Modules Overview

### 1. Ingestion
Handles data source integration and pipeline setup.
- **Complexity Factors**: Data source types, volume, frequency, transformation logic
- **Base Costs**: From $5K (Simple) to $50K+ (Enterprise)

### 2. Datasets
Data organization, storage, and management.
- **Complexity Factors**: Number of datasets, relationships, schema complexity, governance
- **Base Costs**: From $3K (Simple) to $40K+ (Enterprise)

### 3. BI Reports
Analytics, dashboards, and reporting infrastructure.
- **Complexity Factors**: Report types, data sources, visualization complexity, interactivity
- **Base Costs**: From $4K (Simple) to $45K+ (Enterprise)

## Usage

1. **Select Modules**: Choose which modules you need (Ingestion, Datasets, BI Reports)
2. **Choose Complexity**: Pick the appropriate complexity level for each module
3. **Set Quantity**: Adjust quantities if needed (for multiple instances)
4. **Add Optional Services**: Select any add-ons (Support, Training, Migration, etc.)
5. **Apply Discounts**: Select annual commitment discount if applicable
6. **View Pricing**: See real-time pricing breakdown
7. **Export Quote**: Download the quote as a text file
